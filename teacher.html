Lesson Objective

The learning object is to help people understand object and class.

I hope the student could understand the abstract idea of objects. Then they are familiar with classes and able to create their own class and object

I know I have achieve my goal when the student be able to create their own class and object.

Important Detail

I believe I should use examples that is easy to understand and related to the users.

I hope the interaction could be very interesting and the students are willing to continue.

This page is for every knowledge level users.

Design

This page will be based on the users gender and provide different interactive exsamples. Also the whole process will mimic a normal life process. The user would be able to type in informations and the function will based on that information generate different effects. Finally, I want the user try to create their own class and object to make sure they understand the concept

To provide live examples for better understanding and vivid explaination
